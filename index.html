    オープンデータカタログ/データセット¶

    全国¶

    • DATA GO JP - 日本政府のオープンデータカタログ (デジタル庁運営)
    • e-Stat - 政府統計の総合窓口
    • e-Govポータル 行政API - 行政機関等が公表するAPIのまとめ

    文化・芸術・教育¶

    • ジャパンサーチ
    • メディア芸術データベース (文化庁)
      • MADB Lab
    • オープンデータセット｜国立国会図書館-National Diet Library
    • ROIS-DS人文学オープンデータ共同利用センター (CODH)
    • 子供の学び応援サイト掲載コンテンツ情報 (文部科学省) - 子供の学び応援サイト (現 たのしくまなび隊) に掲載している一部の学習コンテンツのメタデータ

    環境・国土・地理¶

    全般¶
    • 環境データショーケース - 環境庁のオープンデータカタログ
    • 国土数値情報ダウンロードサービス (国土交通省)
    • PLATEAU (プラトー) - 国土交通省が主導する、日本全国の3D都市モデルの整備・オープンデータ化プロジェクト
    • G空間情報センター (AIGID)
    不動産¶
    • 不動産情報ライブラリ (国土交通省) - 不動産価格（取引価格・成約価格）情報、地価情報などの不動産関連データのダウンロード、APIも提供 (旧 土地総合情報システム)
    大気・気象¶
    • 環境省大気汚染物質広域監視システム（そらまめくん）API
    河川・海洋・水質¶
    • 荒川下流ＧＩＳオープンデータポータル (荒川下流河川事務所)
    • 海しる / 海しるAPI (海上保安庁 海洋状況表示システムMSIL) - 地形・地質、海象、気象、安全、防災、水産、海域利用・保全などに関する海洋情報を提供するGISおよびAPIプラットフォーム
    地質・地盤・地形¶
    • 地質調査総合センター (GSJ) データカタログ (産総研 地質調査総合センター)
    • 国土地盤情報検索サイト KuniJiban (国土交通省) - ボーリング柱状図や土質試験結果等の地盤情報データ
    • 地理地殻活動研究センター公開データ (国土地理院 地理地殻活動研究センター)

    交通・移動・物流¶

    全般¶
    • 公共交通オープンデータセンター (公共交通オープンデータ協議会)
    • Project LINKS (国土交通省) - 国土交通分野のデータ整備・活用・オープンデータ化プロジェクト
    • 交通事故統計情報のオープンデータ (警察庁)
    道路・歩道¶
    • 歩行者移動支援サービスに関するデータサイト (バリアフリー・ナビプロジェクト) (国土交通省)
    航空・宇宙¶
    • CARATS Open Data (国土交通省・要申請) - 定期航空便の時刻・位置等の航跡データ
      • CARATS Open Data 関連情報 (電子航法研究所)

    防災・安全¶

    • 犯罪オープンデータリンク集 (警察庁)
    • 防災情報提供センター (国土交通省)
    • J-SHIS (地震ハザードステーション) (防災科研) - 確率論的地震動予測地図、条件付超過確率、震源断層を特定した地震動予測地図、地盤情報、被災人口、J-SHIS地震・断層コードなど

    観光・貿易・国際交流¶

    • 日本の観光統計データ (日本政府観光局 (JNTO))
    • 国際会議統計 (日本政府観光局 (JNTO)・非オープンデータ)
    • 訪日外客数・出国日本人数データ (日本政府観光局 (JNTO)・ライセンス不明)
    • 海外安全情報オープンデータ (外務省)
    • 観光統計・白書 (観光庁)
    • 手ぶら観光カウンター情報 (国土交通省)
    • ガイド東北オープンデータ (国土交通省 東北地方整備局) - 東北6県の魅力的な施設やスポット情報

    医療・健康・福祉¶

    • サービス付き高齢者向け住宅情報提供システム 全国登録情報データ - (ライセンス不明)
    • 障害福祉サービス等情報公表システムデータのオープンデータ (厚生労働省社会・援護局)
    • NDBオープンデータ (厚生労働省)
    • 介護サービス情報公表システム (厚生労働省)
      • 通いの場のオープンデータ (厚生労働省)
    • 介護DBオープンデータ (厚生労働省) - 2018〜2019年度のみ
    • 歯科保健医療に関するオープンデータ (厚生労働省)

    金融・産業・商業・法人・法務¶

    • EDINET (金融庁) - 金融商品取引法に基づく有価証券報告書等の開示書類に関する電子開示システム
    • 国税庁法人番号公表サイト ダウンロード/Web API - 法人の基本3情報 (商号又は名称/本店又は主たる事務所の所在地/法人番号) のデータセット
    • 女性の活躍推進企業データベース オープンデータ (厚生労働省)
    • 落札実績オープンデータ (デジタル庁 調達ポータル)
    • 水防災オープンデータ提供サービス (国土交通省/河川情報センター・有償)
    • 登記所備付地図データ (法務省)

    農業¶

    • 国際農研 オープンデータ (国際農林水産業研究センター (JIRCAS)) - 「JIRCASの動き」および「現地の動き」、主催・共催するイベント・シンポジウム、国際農研の刊行物、研究者による出張報告書や研究成果情報、調達情報（入札公告）
